# Patrons de Construction en Java

Ce projet implémente plusieurs patrons de construction en Java, y compris :

- Singleton
- Factory Method
- Abstract Factory
- Builder
- Prototype

## Exécution

Pour exécuter chaque exemple, naviguez dans le dossier approprié et compilez le fichier `Main.java` :

```bash
cd src/singleton
javac Main.java
java Main
